# AdShpere React Website

This is a Vite + React website package prepared for GitHub + Vercel deployment.

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub + Vercel

1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Vercel and import the GitHub repository.
4. Framework preset: **Vite**
5. Build command: `npm run build`
6. Output directory: `dist`
7. Deploy.

## Notes

- Replace the placeholder email `hello@adsphere.com` in `src/App.jsx` if needed.
- The site is currently a multi-page-style single app with top navigation switching between sections/pages.
- You can later split these into real routes if you want.
